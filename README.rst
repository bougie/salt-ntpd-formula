Available states
================

.. contents::
    :local:

``ntpd``
------------

Runs the state to install ntpd and configure the common files.

``ntpd.install``
--------------------

Install ntpd

``ntpd.config``
-------------------

Manage ntpd configuration file.

``ntpd.service``
---------------------

Manage the startup and running state of ntpd.
